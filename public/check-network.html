Use this tool to discover your office network's ASN and IP information.

How to Configure Office Settings:

  1. Click the button above while connected to your office network
  2. Copy your ASN number from the results
  3. Go to Supabase SQL Editor
  4. Run this SQL (replace YOUR_ASN with the number from step 2): 
    UPDATE public.office_settings 
SET setting_value = '["YOUR_ASN"]'::jsonb 
WHERE setting_key = 'allowed_asns';
  5. Or use CIDR range instead: 
    UPDATE public.office_settings 
SET setting_value = '["YOUR_IP/24"]'::jsonb 
WHERE setting_key = 'allowed_cidrs';
